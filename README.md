
# 👩🏻‍💻 Isaias Fernandes 

**`Desenvolvedor FullStack`**

Olá! Meu nome é Isaias Fernandes, tenho 21 anos e sou estudante de Ciência da Computação na Universidade Farias Brito. Atualmente, atuo como Desenvolvedor Full Stack na Usiminas (via Blend IT), onde trabalho com tecnologias como Java (Quarkus), Angular, Oracle SQL, além de práticas de DevOps, CI/CD e integrações de sistemas.
Sou apaixonado por tecnologia, desenvolvimento de sistemas e inovação. Tenho experiência na criação de APIs, construção e manutenção de bancos de dados, desenvolvimento de interfaces web e integração de sistemas críticos. Também possuo conhecimento em ferramentas como Azure DevOps, Postman, Kafka e OpenShift.
Tenho como objetivo me desenvolver cada vez mais como profissional, buscando entregar valor real às empresas através da tecnologia, sempre com responsabilidade, dedicação e mentalidade de crescimento.
